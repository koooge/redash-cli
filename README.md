# redash-cli
Client tool for Redash

## Install
Download from [Release page](https://github.com/koooge/redash-cli/releases)

## Usage
```
$ redash config [--profile <value>]
endpoint_url: <Redash's url>
api_key: <Your API key>

$ redash get-querylist
```
